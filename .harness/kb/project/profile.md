# Project Profile — B2B System

**Stack:** web-nestjs-react
**Description:** B2B System is an order-management platform for business-to-business transactions. Centered on `商流ID` as the core business key, it helps order users select the correct delivery destination, contract terms, products, and pricing; create accounts-receivable orders; and seamlessly sync confirmed orders with the core ERP system, D365.
**Team size:** 3-5
**Environment:** localhost-first
