# /caveman-review

Deliver ultra-compressed code review feedback — one line per issue.

## Usage

```
/caveman-review           # review current diff
/caveman-review <file>    # review specific file
```

Output format: `L<line>: 🔴/🟡/🔵 <problem>. <fix>.`
