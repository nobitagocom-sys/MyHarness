# /caveman

Activate caveman communication mode to reduce output tokens ~75% while keeping full technical accuracy.

## Usage

```
/caveman           # full mode (default)
/caveman lite      # drop filler, keep articles
/caveman full      # remove articles, allow fragments
/caveman ultra     # max abbreviation, arrows for logic
/caveman wenyan    # classical Chinese compression
```

## Deactivate

Say "stop caveman" or "normal mode".
